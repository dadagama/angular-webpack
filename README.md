# angular-webpack-jwt
Boilerplate project to start an Angular App
###Contains
* [AngularJS](https://angularjs.org/) 1.5.x
* [Webpack](https://webpack.github.io/) 1.12.x 
* [JWT](http://jwt.io/introduction/) using [angular-jwt](https://github.com/auth0/angular-jwt)
